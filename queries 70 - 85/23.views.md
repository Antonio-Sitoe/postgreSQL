# ✅ **5 Exercícios para Criar VIEW**

---

## **1. Criar uma VIEW de clientes com seus contatos principais**

**Objetivo:** Consolidar informações do cliente e seu contato mais recente.
**Tarefa:**
Crie uma view que mostre:

- nome do cliente
- email do cliente
- número do contacto mais recente (maior `id`)

Utilize `JOIN` + subconsulta para pegar apenas o último contacto.

---

## **2. Criar uma VIEW de clientes VIP com nota acima de 4**

**Objetivo:** Filtrar clientes com nota alta.
**Tarefa:**
Crie uma view chamada `vw_clientes_vip` contendo:

- id
- nome
- nota
- estado

Somente para clientes onde `nota >= 4`.

---

## **3. Criar uma VIEW de saldo completo das contas**

**Objetivo:** Mostrar visão completa de contas com seus donos.
**Tarefa:**
Crie uma view que mostre:

- id da conta
- nome do cliente dono da conta
- saldo
- data de criação da conta

Use JOIN entre `conta` e `cliente`.

---

## **4. Criar uma VIEW de total de lançamentos por categoria**

**Objetivo:** Consolidar valores financeiros.
**Tarefa:**
Crie uma view que mostre:

- nome da categoria
- quantidade de lançamentos
- soma total dos valores

Agrupe por categoria.

---

## **5. Criar uma VIEW de fornecedores confiáveis**

**Objetivo:** Criar lista filtrada por regra de negócio.
**Tarefa:**
Crie uma view chamada `vw_fornecedores_confiaveis` que contenha:

- id
- nome
- nota
- tipo (ex: “bom”, “excelente”, “ruim”) → baseado em CASE

  - nota ≥ 4 → “excelente”
  - 3 ≤ nota < 4 → “bom”
  - nota < 3 → “ruim”
